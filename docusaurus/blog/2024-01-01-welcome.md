Chào mừng bạn đến với **Memory Docs**! 🎉

Đây là hệ thống tài liệu kết hợp giữa [Docusaurus](https://docusaurus.io/) và [MinIO](https://min.io/) - cho phép bạn viết tài liệu và quản lý media một cách dễ dàng.

<!--truncate-->

## Tại sao chọn Docusaurus + MinIO?

### Docusaurus
- Framework mạnh mẽ được phát triển bởi Meta (Facebook)
- Viết tài liệu đơn giản bằng Markdown
- Fast, SEO-friendly, responsive
- Hỗ trợ versioning, i18n

### MinIO
- Object storage tương thích S3
- Self-hosted, không phụ thuộc cloud provider
- Performance cao
- Dễ dàng scale
